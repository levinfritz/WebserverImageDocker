Docker Apache Webserver

Willkommen auf meinem Docker Apache Webserver

Diese Webseite läuft in einem Docker Container mit Apache auf Port 8080. Der Container wurde erfolgreich gestartet und die Webseite ist nun über den Browser erreichbar.

Dieses Projekt demonstriert die Verwendung von Docker zur Containerisierung eines Webservers mit Apache.

Funktionen des Projekts

Docker Container

Die Anwendung läuft in einem isolierten Docker-Container, der einfach zu verwalten und zu skalieren ist.

Apache Webserver

Verwendet den zuverlässigen Apache HTTP Server für das Hosting der Webseite.

Volume Mounting

Webseiten-Dateien und Logs werden in lokalen Verzeichnissen gespeichert und in den Container gemountet.
